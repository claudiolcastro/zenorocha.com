Desabilitar autocorreção no ZSH

Jun 13, 2013

Desde o ano passado estive usando ZSH como shell ao invés do Bash, nativo do Mac OSx. Graças ao oh-my-zsh consegui ganhar uma produtividade muito boa no terminal.

Só que uma coisinha sempre me irritava, a autocorreção.

git clone git@github.com:zenorocha/diveintohtml5.git
zsh: correct 'clone' to 'clones' [nyae]? n

Finalmente consegui desabilitar, por isso a dica de hoje vai para galera que também usa ZSH e não curte essa feature.

Para desabilitar, basta adicionar a última linha no arquivo ~/.zshrc.

source $ZSH/oh-my-zsh.sh
unsetopt correct_all
